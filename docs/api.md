# Rhymee API 仕様書

日本語の韻検索パブリック API。単語を入力すると、同じ母音パターンを持つ単語を返します。

## Base URL

```
https://<your-domain>/api
```

## エンドポイント一覧

| メソッド | パス         | 説明             |
| -------- | ------------ | ---------------- |
| GET      | `/api/rhyme` | 韻を検索する     |

---

## GET /api/rhyme

入力された単語の母音パターンに一致する単語を検索します。

### リクエストパラメータ

| パラメータ | 型     | 必須 | デフォルト | 説明                                         |
| ---------- | ------ | ---- | ---------- | -------------------------------------------- |
| `word`     | string | ✅   | -          | 検索対象の単語                               |
| `mode`     | string | -    | `suffix`   | 検索モード（`exact` / `suffix` / `partial`） |
| `count`    | number | -    | `3`        | 末尾一致時の母音文字数（1〜8）               |
| `limit`    | number | -    | `50`       | 最大取得件数（1〜200）                       |
| `pos`      | string | -    | -          | 品詞フィルタ（例: `名詞`）                   |
| `shuffle`  | string | -    | `false`    | `true` でランダム順に返す                    |

### 検索モードの説明

| モード    | 説明                                       | 用途                             |
| --------- | ------------------------------------------ | -------------------------------- |
| `exact`   | 母音パターンが完全に一致する単語を返す     | 同じ音数で完全に韻を踏む単語    |
| `suffix`  | 母音パターンの末尾N文字が一致する単語を返す | 語尾で韻を踏む単語（最も一般的）|
| `partial` | 母音パターンが部分的に含まれる単語を返す   | 広めに候補を探したい場合        |

**使い分けガイド:**

- ラップや歌詞で語尾の韻を踏みたい → `suffix`（デフォルト）で `count` を調整
- 同じリズム感の単語が欲しい → `exact`
- とにかく多くの候補から選びたい → `partial`

### レスポンス

```json
{
  "input": "東京",
  "reading": "トウキョウ",
  "vowelPattern": "ouou",
  "mode": "suffix",
  "count": 3,
  "matches": [
    {
      "word": "登頂",
      "reading": "トウチョウ",
      "vowelPattern": "ouou",
      "pos": "名詞",
      "score": 99
    }
  ],
  "total": 42
}
```

| フィールド     | 型             | 説明                       |
| -------------- | -------------- | -------------------------- |
| `input`        | string         | 入力された単語             |
| `reading`      | string         | カタカナ読み               |
| `vowelPattern` | string         | 母音パターン               |
| `mode`         | string         | 使用した検索モード         |
| `count`        | number         | 末尾一致の文字数           |
| `matches`      | RhymeMatch[]   | マッチした単語の配列       |
| `total`        | number         | マッチした件数             |

#### RhymeMatch オブジェクト

| フィールド     | 型     | 説明                                                    |
| -------------- | ------ | ------------------------------------------------------- |
| `word`         | string | 表層形                                                  |
| `reading`      | string | カタカナ読み                                            |
| `vowelPattern` | string | 母音パターン                                            |
| `pos`          | string | 品詞                                                    |
| `score`        | number | 韻スコア（0〜100）。100 は同音異義語、それ以外は最大99 |

### cURL リクエスト例

**基本的な検索（suffix モード）:**

```bash
curl "https://<your-domain>/api/rhyme?word=東京"
```

**完全一致モード:**

```bash
curl "https://<your-domain>/api/rhyme?word=東京&mode=exact"
```

**末尾4文字一致 + 名詞のみ:**

```bash
curl "https://<your-domain>/api/rhyme?word=東京&mode=suffix&count=4&pos=名詞"
```

**ランダム順で取得:**

```bash
curl "https://<your-domain>/api/rhyme?word=東京&shuffle=true"
```

**取得件数を制限:**

```bash
curl "https://<your-domain>/api/rhyme?word=サッカー&limit=10"
```

## エラーレスポンス

すべてのエラーは以下の形式で返されます:

```json
{
  "error": "エラーメッセージ"
}
```

| ステータスコード | 説明                                     |
| ---------------- | ---------------------------------------- |
| `400`            | リクエストパラメータが不正               |
| `500`            | 形態素解析に失敗（サーバー内部エラー）   |
| `503`            | データベース接続に失敗                   |

### エラー例

**word パラメータ未指定（400）:**

```json
{
  "error": "パラメータ 'word' は必須です"
}
```

**不正な mode 指定（400）:**

```json
{
  "error": "パラメータ 'mode' は exact, suffix, partial のいずれかです"
}
```
